as Eclipse plugin.

It is released under the LGPL (see the file LICENSE.TXT, which should be located
in the same directory as this file). If you didn't recieve a copy of the license
file, please contact us: contact@knime.org



This plugin makes use of the R client code which is under the copyright of its 
respective authors. The library can be found in the lib directory, accompanied
by its source code. It can also be downloaded at 
http://stats.math.uni-augsburg.de/Rserve
The R project has its home page at http://www.r-project.org/. 

Eclipse is TM of the Eclipse Foundation ("http://www.eclipse.org").

Quickstart guide:
- Extract the archive (you did already when you read this)
- Copy the plugins/org.knime.ext.r_1.0.0 folder to your KNIME
  installation's plugins directory.
- Download and install the R Server, http://stats.math.uni-augsburg.de/Rserve
- Download and install the http://www.r-project.org
- Restart eclipse to register the newly added plugin.
